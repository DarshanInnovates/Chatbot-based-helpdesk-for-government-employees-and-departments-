# Chatbot-based-helpdesk-for-government-employees-and-departments-
This project implements an AI-powered chatbot system designed to streamline communication between government employees and departments. The chatbot provides instant responses to FAQs, assists with HR/IT support queries, automates ticket generation, and routes unresolved issues to relevant officials.
